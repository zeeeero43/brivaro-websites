## üìù Notes

### Animation Performance Tips
- Use `transform` and `opacity` only (GPU-accelerated)
- Add `will-change` for elements that will animate
- Remove `will-change` after animation completes
- Use `requestAnimationFrame` for JS animations
- Debounce scroll event listeners
- Use Intersection Observer for scroll triggers

### Content Strategy
- Keep headlines short and impactful
- Use bullet points for easy scanning
- Include social proof throughout
- Clear CTAs on every section
- Trust signals (DSGVO, SSL, etc.)
- Focus on benefits, not features

### Mobile-First Considerations
- Simplified animations on mobile
- Larger touch targets (min 44x44px)
- Reduced content where appropriate
- Optimized images for mobile bandwidth
- Fast load times (< 3s on 4G)

### Accessibility Checklist
- Keyboard navigation works everywhere
- Focus indicators visible
- Color contrast meets WCAG AA
- Alt text on all images
- ARIA labels on interactive elements
- Respects prefers-reduced-motion
- Screen reader tested
